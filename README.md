🐱 Magicats 🧙

Jogo 2D desenvolvido na Unity utilizando C# como Projeto Integrador do curso Técnico em Informática do SENAC.

## 🚀 Tecnologias

- C#
- Unity
- Programação Orientada a Objetos
- Git

---

## 📚 Conceitos aplicados

- Encapsulamento
- Herança
- Polimorfismo
- Classes e Objetos
- Métodos
- Tratamento de colisões
- Controle de estados do jogo

---

## 🛠 Funcionalidades

- Movimentação do personagem
- Sistema de vidas
- Sistema de pontuação
- Colisões
- Fases
- Interface do usuário

---

## 💡 O que aprendi

Este foi meu primeiro projeto de maior porte utilizando C# e Unity. Durante o desenvolvimento, pude aplicar conceitos de Programação Orientada a Objetos, trabalhar em equipe, desenvolver lógica de programação e entender a importância da organização do código em um projeto real. Revisitar esse projeto hoje me permite perceber minha evolução e reforçar minha base em desenvolvimento de software.
